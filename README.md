A modern movie discovery web application built with React, Redux, and Tailwind CSS. Browse movies, TV shows, and explore actor information with an intuitive interface powered by The Movie Database (TMDB) API.

## Features

- **Movie & TV Show Discovery** - Browse trending, popular, and categorized content
- **Detailed Information** - View comprehensive details about movies, TV shows, and cast members
- **Search & Filter** - Find movies and shows by name and genre
- **Person Details** - Explore actor profiles and filmography

## Tech Stack

- **Frontend Framework** - React 18.3
- **State Management** - Redux Toolkit & Redux
- **Styling** - Tailwind CSS
- **Build Tool** - Vite
- **HTTP Client** - Axios
- **Routing** - React Router DOM
